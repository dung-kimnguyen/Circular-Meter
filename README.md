# Features!
  - Circular ring slider
  - Customizable slider color
  - Customizable slider knob
  - Customizable sider end point buttons
  - Adjust slider by editing center label
  - Dynammic values of slider


# Requirements
  - iOS 10.0+
  - Xcode 9+
